# Weather Dashboard

A React weather dashboard built with Vite, using the OpenWeatherMap API.

## Features
- Current weather conditions (temperature, humidity, wind, visibility, pressure)
- 5-day forecast
- Celsius / Fahrenheit toggle
- Search history (last 5 cities, persisted in localStorage)

## Setup
1. Clone the repo
2. Run `npm install`
3. Replace `YOUR_OPENWEATHER_API_KEY` in `App.jsx` with your key from [openweathermap.org](https://openweathermap.org/api)
4. Run `npm run dev`

## Tech Stack
- React 18
- Vite
- OpenWeatherMap API
